SQL - Covid Deaths analysis

Dataset & Code @GitHub.GitHub

Skills: Microsoft SQL (Queries, sub-queries, CTE (common table expressions), views, joins, temporary tables), Integrating SQL with Pyton, Jupyter notebooks, Pandas, Plotly, Numpy , PowerBi for Dashboards

Use Case:Analyse the Covid 19 deaths on global scale using Microsoft SQL server.

Challenge:Sometimes data analysis involves working with millions of rows of data available in a database. Extracting this huge amount of data into CSV is not recommanded in this situation .This is where we can use PYODBC driver to connect python to SQL server and read the data using SQL queries.

Sample Data:

Dataset used is from "Our World in Data" website can be found here. The data is divided into two parts where one table contains the covid death details and other contains the vaccinations details. Number of rows in both table is around 360 thousand rows.
